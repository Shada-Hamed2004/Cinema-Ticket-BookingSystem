# Cinema Ticket Booking System

A simple Java console application for booking cinema tickets.

Users can view available movies, choose a movie, select Regular or VIP ticket, 
and see ticket details with dynamic pricing.

Features:
- Display movie list
- Book Regular or VIP ticket
- Calculate price based on ticket type
- Show ticket information

Technologies: Java, OOP (Inheritance, Polymorphism, Abstract Classes), ArrayList, Exception Handling
